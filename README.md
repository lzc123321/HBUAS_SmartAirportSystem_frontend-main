# BUAA_SE_2023_SmartAirportSystem_frontend

## 启动vue项目

使用 `npm run serve` 命令





## 插件使用

使用了Element Plus（内含组件/图标等）

网址：[一个 Vue 3 UI 框架 | Element Plus (element-plus.org)](https://element-plus.org/zh-CN/)
